Miner

Description

Miner is a lightweight Minecraft Server manager CLI tool. The benefit of using Miner is that the user never has to worry about downloading anything ever again. Including modding systems.

Miner uses Docker in the background to manage and maintain various versions of Minecraft Servers and Modding systems like Forge and Craftbukkit.

Minecraft Versions

I'm trying to support new and older versions as they get available. The docker instances can be found in the link below where you can see what Minecraft versions are available.
